# AI 电商选品多Agent系统
基于大语言模型 + LangChain 构建的自动化电商选品AI Agent，自动完成「多平台数据爬取→竞品分析→趋势研判→输出选品报告」全流程，帮助中小电商卖家节省选品时间，挖掘蓝海细分赛道。

## 核心解决痛点
传统电商选品需要卖家手动到多个平台爬数据、查热度、分析竞品，整个流程耗时几小时甚至数天，中小卖家没有工具支持很容易因为信息差错过风口，或者切入红海赛道亏本。

本项目通过多Agent分工自动完成全流程选品工作，把选品时间压缩到几分钟。

## 功能特点
- 🤖 **多Agent协作**​：数据爬取/竞品分析/趋势研判自动调度，无需人工分步操作
- 📊 **结构化输出**​：自动生成包含赛道总结、机会点、切入建议、风险提示的完整选品报告
- 🛠️ **可扩展**​：支持对接真实电商平台API，可自定义拓展品类、新增平台数据
